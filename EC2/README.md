# EC2

This folder contains documentation and resources for Amazon EC2 (Elastic Compute Cloud) labs.

## Contents

- **Lab Documentation.md** - Detailed lab documentation covering EC2 instance management, configuration, and security
- **assets/** - Screenshots and diagrams referenced in the lab documentation

## Overview

This lab provides hands-on experience with:
- EC2 Instance Launch and Configuration
- Instance Types and Selection
- Security Groups and Network Security
- User Data Scripts
- Stop Protection Features
- System Logs and Monitoring

For detailed lab instructions and documentation, see [Lab Documentation.md](Lab Documentation.md)
