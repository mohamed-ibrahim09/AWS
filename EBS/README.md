# EBS

This folder contains documentation and resources for Amazon EBS (Elastic Block Store) labs.

## Contents

- **Lab Documentation.md** - Detailed lab documentation covering EBS volume management, snapshots, and data persistence
- **assets/** - Screenshots and diagrams referenced in the lab documentation

## Overview

This lab provides hands-on experience with:
- EBS Volume Creation and Configuration
- Volume Attachment to EC2 Instances
- Snapshot Management
- Data Backup and Restore
- Volume Performance and Types

For detailed lab instructions and documentation, see [Lab Documentation.md](Lab Documentation.md)
