# VPC

This folder contains documentation and resources for Amazon VPC (Virtual Private Cloud) labs.

## Contents

- **Lab Documentation.md** - Detailed lab documentation covering VPC networking, subnets, and network architecture
- **assets/** - Screenshots and diagrams referenced in the lab documentation

## Overview

This lab provides hands-on experience with:
- VPC Creation and Configuration
- Subnet Design and Management
- Availability Zones
- Network ACLs and Security Groups
- VPC Peering and Connectivity

For detailed lab instructions and documentation, see [Lab Documentation.md](Lab Documentation.md)
